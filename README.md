- `Criar tarefa através de um Modal.`
- `Remover Tarefa.`
- `Campo para editar todas informações de uma tarefa.`
- `Opção de alterar o estado de uma tarefa para "Completa" ou "Incompleta."`

- `Duas opções de tema: Claro e Escuro.`
- `Barra de pesquisa.`
